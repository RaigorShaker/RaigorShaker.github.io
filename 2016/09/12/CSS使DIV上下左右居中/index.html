CSS div上下左右居中

CSS使DIV上下左右居中

标签（空格分隔）：css

div上下左右居中方法

  • 方法一

    最直观的方法，可以给div2设置margin-left和margin-top来使div2上下左右居中，margin-left值为外层div的宽度的1/2 - 内部div宽度的1/2，margin-top的值同理，只不过取得基准为div的高度。
    在此例中，div2的margin-left和margin-top的值应该都为30px，修改后如下：

  • 方法二
    最简单的方法，将外层的div1的display属性设置成table-cell，并添加vertical-align:middle样式，内部的div2的样式添加margin:auto即可实现上下左右居中，因为表格样式里自动会包含居中对齐的样式，所以此种方法最简便，并且不需要额外的计算。推荐使用。

  • 方法三
    可以对div2采用绝对定位的方式来处理，前提是要相对于div1来绝对定位，所以需要给div1添加position:relative样式，并且将div2的left、right、bottom、top属性都设置为0，另外添加margin:auto属性，就可以使div2上下左右居中了。
